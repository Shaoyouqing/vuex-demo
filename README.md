## vuex:
```

state:驱动应用的数据源
getters:为组件提供经过处理的数据源
mutations:用来更改state的唯一标准方式
actions:组件通过调用actions来提交mutation
modules:每个module都有自己的state、getters、mutations、actions实现大型应用中业务模块数据的分治

```
